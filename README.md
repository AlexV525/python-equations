# python-equations

Small collection of Tkinter-based Python calculators and converters for common math tasks.

## Requirements

- Python 3.x
- Tkinter (bundled with most Python installs; on some Linux distros install `python3-tk`)

## Running

Each tool is a standalone script. Run any file with Python:

```bash
python "Quadratic Equation/Quadratic Equation.py"
```

Paths with spaces should be quoted as shown above.

## Available tools

- `DT 105906(4).py` – simple calculator (add/subtract/multiply/divide) with feedback prompts
- `DT example.py` – square a number
- `Area/Area.py` – area of triangle, rectangle, circle, and trapezoid
- `Convert/Convert.py` and `Convert/cv.py` – km→mi, °C→°F, degrees→radians
- `Divisible/Divisible.py` – find list values divisible by an input
- `factorial number/factorial.py` – factorial calculator
- `Leap year/Leap year.py` – leap year checker
- `Mean/Mean, Median, Mode, Std, Variance.py` – basic statistics calculator
- `multiplication/multiplication.py` – multiplication table generator
- `NumConvert/NumConvert.py` – decimal to binary/octal/hex converter
- `Quadratic Equation/Quadratic Equation.py` – quadratic equation solver
- `Square/Square & Sqrt & Cube & Cbrt.py` – square/square root/cube/cube root
- `color.py` – Tkinter color palette viewer

## Notes

- Most inputs expect numeric values; some tools accept comma-separated lists as described in their UI.
- This repo contains `.docx` files with additional descriptions for some exercises.
